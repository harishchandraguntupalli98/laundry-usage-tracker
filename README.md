# Laundry Usage Tracker

A simple Java full-stack web application to track washer and dryer usage in a shared house.

## Tech Stack
- Java
- Spring Boot
- HTML, CSS, JavaScript
- REST APIs

## Features
- View washer and dryer status
- Mark machine as in use
- Release machine after use

## How to Run
1. Run Spring Boot application
2. Open index.html in browser
